# Translate-cli

Command line translator, which uses [Ligva Translate](https://github.com/thedaviddelta/lingva-translate), Google translator front-end.

```
$ python translate-cli.py world
auto (en) >> cs

svět

[wərld]
Other translations:
        noun
          - svět
          - země
          - lidstvo
          - společnost
```
